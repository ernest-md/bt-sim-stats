# OP Balatro

Starter en `Phaser + TypeScript + Vite` para construir un clon de Balatro inspirado en One Piece Card Game.

## Incluye

- Estructura base con escenas `boot`, `menu` y `run`
- Prototipo jugable con mano, mazo, descarte, energia y limite de jugadas
- Cartas de ejemplo con facciones, tags y sinergias
- Sistema de puntuacion estilo `chips x mult`

## Arranque

```bash
npm install
npm run dev
```

## Siguientes pasos recomendados

1. Sustituir las cartas mock por un pool real de lideres, personajes y eventos.
2. Separar la logica de scoring en reglas modulares para jokers/relics.
3. Añadir tienda, recompensas post-ronda y mapa de progresion.
4. Persistir runs y coleccion de cartas.
